Le jeu du morpion-moulin

Afin de nous initier à l'orienté-objet en JAVA, il nous a été demandé de construire une application java permettant d'effectuer un jeu du morpion qui reprend le concept du jeu du moulin. Trois modes de jeu sont proposés :

Le joueur a le choix sur le fonctionnement du joueur simulé par l'ordinateur.

  • Soit par choix aléatoires

L'algorithme du minimax est l'un des algorithmes basiques en matière d'Intelligence Artificielle.

Une application web créatrice de QCMs

Ce projet au début de ma seconde année fut un tournant pour ma part, car pour la première fois, à l'aide d'une équipe de 4 personnes, on s'est attaqué à la partie back-end pour la conception d'une application web.
Cette application avait pour but de générer des QCMs pour un groupe de personnes (d'étudiants par exemple). Il a fallu distinguer les différentes services que pouvait composer cette application, et les implémenter en respectant le pattern MVC (Modèle Vue Contrôle) :

  • La partie Modèle de l'application contient toutes les ressources stockées dans une base de données
  • La partie Contrôle intéragit entre le Modèle et la Vue. Elle interroge d'abord le Modèle pour récupérer des données, pour ensuite les afficher dans la partie Vue.
  • La partie Vue est la partie Front-End de l'application Web. Le contenu affiché est le travail du Contrôle.

Un site touristique sur l'Alsace

Toujours sur le thème du développement web mais en Front-End cette fois-ci, on nous a demandé de développer un site sur une région, encore une fois avec le CMS WIX. Nous étions une équipe de 4, et notre choix s'est porté sur l'Alsace en raison de sa grande culture.

Après le développement sur CMS, on nous a demandé de refaire le site "from scratch". C'est-à-dire de le refaire en "dur", en utilisant les outils HTML5/CSS3 tout en évitant le JavaScript !

Cette seconde partie du projet m'a permis d'énormément progresser en ce qui touche au Front-End : j'ai réussi à assimiler des technologies que propose le CSS3 notamment avec le système de grilles du CSS Grid. C'est grâce à cet exercice que j'ai pu réaliser ce portfolio !
